# Personal Portfolio Website

This project is a **Personal Portfolio Website** developed using **HTML and CSS**.  
The website showcases my profile, technical skills, projects, education, research publications, and contact information in a professional and responsive layout.

This portfolio was created as part of the **CodSoft Web Development Internship Task 2**.

## Features

• Responsive design for desktop, tablet, and mobile devices  
• Professional dark-themed user interface  
• Hero section with profile image and introduction  
• About section describing background and interests  
• Technical skills section  
• Technologies and tools used section  
• Projects showcase section  
• Experience section  
• Research publications section  
• Education section with card layout  
• Achievements and career goals section  
• Contact information section  
• Resume download option
